## 目录介绍
```
├─apps //项目目录
├─packages // 公共资源目录
│ ├─layout // 项目公共布局
│ ├─static // 公共组件
│ ├─types // 公共类型文件
│ └─utils // 公共工具方法
├─public // 公共静态资源
```

## 项目运行

```bash
git clone https://gitee.com/GUANGHUIJs/gh-admin.git
cd gh-admin
pnpm i
cd apps
cd test
pnpm dev
````
