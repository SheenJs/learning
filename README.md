看以前的学习写的代码,看着页面效想不起代码怎么写的,所以干脆搞一个代码展示组件,一边看效果一边看源代码。还把写过的demo统一存放,看的时候更方便,若以后遇到相同的需求也方便查看源码。

## Git 提交规范
- 🍐 `feat` 增加新功能
- 🍐 `fix` 修复问题/BUG
- 🍐 `style` 代码风格相关无影响运行结果的
- 🍐 `refactor` 重构
- 🍐 `docs` 文档/注释
- 🍐 `types` 类型定义文件更改
- 🍐 `wip` 开发中
- 🍐 `build` 打包提交
